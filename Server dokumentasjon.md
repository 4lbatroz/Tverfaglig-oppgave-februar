Logg CT oppsett

Vi skal sette opp to servere.
1 Web server og 1 Database server,
Server oppsettet er helt likt uten om hvilken CT template du velger på første side i wizarden

1. vi gir CT en ID som er unik og et navn som er relevant til hva den skal gjøre og et passord du husker.<img width="452" alt="image" src="https://github.com/4lbatroz/Tverfaglig-oppgave-februar/assets/134390839/c26a65bc-957b-4225-8ecf-3ba181d17d5a">

2. På template velger vi enten en database teplate som MySQL eller en template vi kan bruke som webserver som Ubuntu.<img width="452" alt="image" src="https://github.com/4lbatroz/Tverfaglig-oppgave-februar/assets/134390839/9aef73fa-d8df-4403-aab5-9760a480e6ca">

3. For at vi skal kunne kjøre litt tyngere program setter vi disk størelsen til 256.<img width="229" alt="image" src="https://github.com/4lbatroz/Tverfaglig-oppgave-februar/assets/134390839/b284f73b-6f66-4609-ab79-3524229a966b">

4. her vil vi ha 2 cores så vi har en litt sterkere CT.<img width="452" alt="image" src="https://github.com/4lbatroz/Tverfaglig-oppgave-februar/assets/134390839/0b898753-f203-4270-8948-a0379deea8a1">

5. memory og swap endrer vi til 2048.<img width="452" alt="image" src="https://github.com/4lbatroz/Tverfaglig-oppgave-februar/assets/134390839/dcb25f1e-4b02-435b-9108-c94a3947975f">

6. Network ender vi bare IPv4 til DHCP<img width="452" alt="image" src="https://github.com/4lbatroz/Tverfaglig-oppgave-februar/assets/134390839/54976a5c-5aef-4aee-8581-ad39dd4756ea">

7. her endrer vi ingen ting<img width="452" alt="image" src="https://github.com/4lbatroz/Tverfaglig-oppgave-februar/assets/134390839 og /151e8087-fd2e-4a26-a929-39849005b608">

8. Når du har sjekket at alt er riktig så trykker du finish <img width="452" alt="image" src="https://github.com/4lbatroz/Tverfaglig-oppgave-februar/assets/134390839/4e0496a0-cc61-4dbe-b0b8-613eebc5cd5d">
